## Technologies

**MongoDB** database  
**Node** & **Express** server & api routing  
**React** & **Redux** front-end  
**Bootstrap 4** styling

**Passport** and **JSON Web Tokens** for login authentication  
**NODE ACL** for access control and authorization  
**bcrypt** for password hashing
